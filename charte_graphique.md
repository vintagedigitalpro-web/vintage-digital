Nous privilégions un univers clair, aéré et chaleureux pour inspirer une confiance immédiate.

| Couleur | Code HSL | Code HEX | Rôle principal |

## 4. Typographie & Règles de Mise en Page

* **Polices de caractères** :
  * Titres majeurs : **Outfit** (Extra-Bold, espacement serré `-0.02em` pour un look affirmé).
  * Texte courant : **Inter** (Regular, hauteur de ligne `1.6` pour un confort de lecture optimal).
* **Mise en page & Contrastes** :
  * Grands espaces blancs épurés, ombres portées extrêmement douces, absence totale de fioritures agressives.
  * Bordures fines et dorées ou ardoise ultra-légères pour délimiter les structures sans agressivité.
